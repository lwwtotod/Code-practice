查看`makefile`文件 执行`make`命令

使用`yarn`或`npm`安装依赖

安装完依赖之后执行`dev`命令

```nodejs
yarn bootstrap

yarn dev
```

运行完 dev 命令后在`build`文件夹下`webpack.demo.js`查看端口号

```js
  devServer: {
    host: '0.0.0.0',
    port: 8085,
    publicPath: '/',
    noInfo: true
  },
```

打开[localhost:8085](localhost:8085)预览项目
